Описание файлов:
верхняя планка волк7 - без разделения, для болього принтера (длина около 300 мм.)
верхняя планка волк7_1,2,3 - планка разделена на 3части для печати на маленьких принтерах(у меня 134х80)
аналогично:
рамка_низ_7 - целая рамка
рамка_низ_7_л - левая часть
рамка_низ_7_пр - правая часть

материал для печати: смола м58
Для установки панели в корпус Василия, необходимо вкорпусе спилить внутренние кромки 
слева, снизу, справа, примерно на 3 мм.

Основа панели - листовой алюминий 2 мм, программу для чпу с чертежами выложу позже.
Ось нижних кнопок - пруток 3мм (можно очищенный электрод)

Основной энкодер крепится штатно, на плату, тип штока - D

Высота кнопок передней панели должна быть 12 мм.
